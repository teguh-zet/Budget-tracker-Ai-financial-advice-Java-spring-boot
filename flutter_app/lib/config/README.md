# Konfigurasi

File konfigurasi untuk aplikasi Flutter Budget Tracker.

## api_config.dart

Konfigurasi untuk API endpoint. Ubah `baseUrl` sesuai dengan URL backend Anda.

Default: `http://localhost:5001/api/v1`

Untuk development dengan emulator Android, gunakan:
- `http://10.0.2.2:5001/api/v1` (Android Emulator)
- `http://localhost:5001/api/v1` (iOS Simulator / Web)

Untuk device fisik, gunakan IP address komputer Anda:
- `http://192.168.x.x:5001/api/v1`
